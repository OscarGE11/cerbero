# Cerbero — Comandos para @BotFather
#
# 1. Abre @BotFather en Telegram
# 2. /setcommands → selecciona tu bot
# 3. Pega el bloque de abajo (sin estas líneas de comentario)
#
# 4. Configura el Menu Button:
#    /setmenubutton → tu bot → Web App
#    URL: https://tu-dashboard.vercel.app/telegram
#
# Límite BotFather: descripción ≤256 caracteres por comando

start - Abrir Cerbero y ver instrucciones
login - Web App para registrarte y vincular cuenta
link - Vincular con código OTP del dashboard (alternativa)
unlink - Desvincular tu Telegram de Cerbero
add - Añadir gasto o ingreso (Web App)
last - Ver los últimos 5 movimientos
month - Resumen del mes actual
dashboard - Abrir el panel web
